# to check the node version

```
node --varsion
```

# to initialize the firebase console

```
firebase init
```

# to start the server

```
npm run server
```

# to create React project using npm and yarn

```
npm create react-app folder_name
```
```
yarn create react-app folder_name
```

# to start the client

```
yarn start
```
